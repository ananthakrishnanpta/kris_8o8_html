Advanced HTML

Advanced HTML topics include:

  • SVG (Scalable Vector Graphics) for vector graphics: <svg>
  • Canvas for drawing graphics via scripting (usually JavaScript): <canvas>
  • Web Storage API: Local Storage and Session Storage
  • HTML5 APIs: Geolocation, Drag and Drop, Web Workers
